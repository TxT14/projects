## ERGON

A minimal Trello-style task manager for organizing work.

**Features:**
- Drag & drop cards between columns
- Drag columns to reorder
- Priority levels (high, med, low)
- Custom tags with color coding
- Filter by priority and tags
- Dark minimalist theme
- Local storage persistence

**Tech:** Vanilla HTML/CSS/JavaScript - no frameworks, no dependencies.

**Usage:** Open `ergon/index.html` in your browser.
